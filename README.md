LibraryOS - Kütüphane Yönetim Sistemi

LibraryOS, modern ve kullanıcı dostu bir arayüze sahip, PHP ve MySQL tabanlı bir kütüphane yönetim sistemidir. Bu proje, kütüphanelerdeki kitap, üye ve ödünç alma/iade işlemlerinin verimli bir şekilde yönetilmesini sağlamak amacıyla geliştirilmiştir.

Özellikler

📊 Yönetim Paneli (Dashboard): Toplam kitap türü, toplam stok adedi gibi önemli istatistiklerin anlık olarak görüntülendiği bir ana sayfa.

📚 Kitap Yönetimi (CRUD):

Yeni kitap ekleme (kapak resmi yükleme dahil).

Mevcut kitapları listeleme, arama ve filtreleme.

Kitap bilgilerini güncelleme.

Kitapları sistemden silme.

👤 Üye Yönetimi:

Sistemdeki tüm üyeleri listeleme.

Gerektiğinde üyeleri sistemden silme.

🔄 Ödünç/İade İşlemleri:

Tüm işlem geçmişini (ödünç alma, iade, gecikenler) görüntüleme.

İşlemlerin durumunu (iade edildi, ödünç alındı vb.) güncelleme.

🔐 Güvenlik:

Session tabanlı kullanıcı ve admin yetkilendirme.

SQL Injection saldırılarına karşı Prepared Statements kullanımı.

📱 Duyarlı Tasarım (Responsive Design): Masaüstü, tablet ve mobil cihazlarla tam uyumlu, modern ve kullanışlı arayüz.


Kullanılan Teknolojiler

Backend: PHP

Veritabanı: MySQL

Frontend: HTML5, CSS3, JavaScript (Vanilla JS)

Lokal Sunucu Ortamı: XAMPP / WAMP
